15.1 – 15.2

Qo'sh Integrallar — Double Integrals

Nazariya

Qo'sh integral ta'rifi (§15.1)

Bitta o'zgaruvchili integralda biz [a, b] kesmani n ta kichik kesmaga bo'lib Riman summasi hosil qilamiz. Qo'sh integralda esa to'rtburchak R = [a,b] × [c,d] ni kichik to'rtburchaklarga bo'lamiz.

Har bir kichik to'rtburchak Rij da namunaviy nuqta (x*ij, y*ij) tanlaymiz va Riman summasini hosil qilamiz:

Qo'sh Riman summasi
∑ᵢ∑ⱼ f(x*ᵢⱼ, y*ᵢⱼ) ΔA
Qo'sh integral ta'rifi (limiti)
∬_R f(x,y) dA = lim_{m,n→∞} ∑ᵢ∑ⱼ f(x*ᵢⱼ, y*ᵢⱼ) ΔA
Geometrik Ma'no

Agar f(x,y) ≥ 0 bo'lsa, qo'sh integral z = f(x,y) sirt ostidagi hajmni ifodalaydi.

Teorema

Fubini Teoremasi (§15.1)

Agar f R = [a,b] × [c,d] da uzluksiz bo'lsa, u holda qo'sh integralni takrorlangan integral sifatida hisoblash mumkin:

Fubini Teoremasi
∬_R f(x,y) dA = ∫_a^b [∫_c^d f(x,y) dy] dx = ∫_c^d [∫_a^b f(x,y) dx] dy

Bu degani: avval ichki integralni (masalan, y bo'yicha) hisoblaymiz, so'ng tashqi integralni (x bo'yicha). Tartib muhim emas — natija bir xil.

Kitob Misoli 15.1 — Paraboloid hajmi

z = 16 − x² − 2y², R = [0,2] × [0,2] — hajmni taxminlang

m = n = 2 bo'linish bilan yuqori o'ng burchak nuqtalarini namunaviy sifatida olamiz:

  1. ΔA = 1 (har bir to'rtburchak maydoni)
  2. f(1,1) = 16 − 1 − 2 = 13 (R₁₁ to'rtburchagi)
  3. f(1,2) = 16 − 1 − 8 = 7 (R₁₂)
  4. f(2,1) = 16 − 4 − 2 = 10 (R₂₁)
  5. f(2,2) = 16 − 4 − 8 = 4 (R₂₂)
  6. V ≈ 13(1) + 7(1) + 10(1) + 4(1) = 34 (aniq javob: V = 48)
Kitob Misoli — Fubini bilan hisoblash

∬_R (x − 3y²) dA, R = {(x,y) | 0 ≤ x ≤ 2, 1 ≤ y ≤ 2} ni toping

  1. Avval y bo'yicha integrallaymiz (x = const):
    ∫₁² (x − 3y²) dy = [xy − y³]₁² = (2x − 8) − (x − 1) = x − 7
  2. So'ng x bo'yicha integrallaymiz:
    ∫₀² (x − 7) dx = [x²/2 − 7x]₀² = (2 − 14) − 0 = −12
Javob: −12
15.2 — Nazariya

Umumiy Soha D ustidagi qo'sh integral

Endi D ixtiyoriy soha bo'lsin. Ikkita tur mavjud:

I-tur soha (x bo'yicha cheklangan)
D: a≤x≤b, g₁(x)≤y≤g₂(x) ∬_D f dA = ∫_a^b ∫_{g₁(x)}^{g₂(x)} f(x,y) dy dx
II-tur soha (y bo'yicha cheklangan)
D: c≤y≤d, h₁(y)≤x≤h₂(y) ∬_D f dA = ∫_c^d ∫_{h₁(y)}^{h₂(y)} f(x,y) dx dy
Kitob Misoli 15.2 — Paraboloid hajmi umumiy sohada

z = x² + y², D = {y = 2x va y = x² orasidagi soha} — hajmni toping

  1. Kesishish nuqtasini topamiz: 2x = x² → x = 0 va x = 2. Demak 0 ≤ x ≤ 2, x² ≤ y ≤ 2x.
  2. Integral yozamiz (I-tur):
    V = ∫₀² ∫_{x²}^{2x} (x² + y²) dy dx
  3. y bo'yicha integrallaymiz:
    ∫_{x²}^{2x} (x² + y²) dy = [x²y + y³/3]_{x²}^{2x} = (2x³ + 8x³/3) − (x⁴ + x⁶/3)
  4. x bo'yicha integrallaymiz:
    = ∫₀² (14x³/3 − x⁴ − x⁶/3) dx = [14x⁴/12 − x⁵/5 − x⁷/21]₀² = 216/35
Javob: V = 216/35
15.3

Polar Koordinatlar

Nazariya

Polar koordinataga o'tish

Agar D doira yoki halqasimon soha bo'lsa, polar koordinatalar juda qulay. x = r cosθ, y = r sinθ o'rniga qo'yganda muhim: dA = r dr dθ — "r" ni unutmang!

Polar koordinatada qo'sh integral
∬_D f(x,y) dA = ∫_α^β ∫_{h₁(θ)}^{h₂(θ)} f(r cosθ, r sinθ) · r dr dθ
⚠️

Eng ko'p kechadigan xato: dA = dr dθ deb yozish. To'g'ri: dA = r dr dθ. "r" — Yakobian koeffitsienti.

Polar misol — Doira ustidagi integral

∬_D (3x + 4y²) dA, D = {x² + y² ≤ 4, yuqori yarim doira}

  1. Chegaralar polar da: 0 ≤ θ ≤ π, 0 ≤ r ≤ 2
  2. x va y ni almashtirish: 3x + 4y² = 3r cosθ + 4r² sin²θ
  3. Integral yozamiz:
    ∫₀^π ∫₀² (3r cosθ + 4r² sin²θ) · r dr dθ = ∫₀^π ∫₀² (3r² cosθ + 4r³ sin²θ) dr dθ
  4. r bo'yicha:
    = ∫₀^π [r³ cosθ + r⁴ sin²θ]₀² dθ = ∫₀^π (8 cosθ + 16 sin²θ) dθ
  5. θ bo'yicha:
    = [8 sinθ + 8θ − 4 sin 2θ]₀^π = 0 + 8π − 0 = 8π
Javob: 8π
Polar misol — Yuza maydoni

Doiraning yuzasi A(D) ni polar koordinatada hisoblang, D: x² + y² ≤ a²

  1. f(x,y) = 1 deb olamiz: ∬_D 1 dA = A(D)
  2. Chegaralar: 0 ≤ θ ≤ 2π, 0 ≤ r ≤ a
  3. Hisoblash:
    ∫₀^{2π} ∫₀^a r dr dθ = ∫₀^{2π} [r²/2]₀^a dθ = ∫₀^{2π} a²/2 dθ = πa²
Javob: A = πa² ✓ (to'g'ri!)
15.6

Uch O'lchovli Integrallar — Triple Integrals

Nazariya

Triple integral ta'rifi (§15.6)

B = [a,b] × [c,d] × [r,s] parallelepiped bo'lsin. Uni l×m×n kichik parallelepiped Bijk ga bo'lamiz, ΔV = ΔxΔyΔz.

Triple integral
∭_B f(x,y,z) dV = lim_{l,m,n→∞} ∑ᵢ∑ⱼ∑ₖ f(x*ᵢⱼₖ, y*ᵢⱼₖ, z*ᵢⱼₖ) ΔV
Fubini Teoremasi (triple)
∭_B f(x,y,z) dV = ∫_r^s ∫_c^d ∫_a^b f(x,y,z) dx dy dz

6 ta integrallash tartibi mavjud — bari bir xil natija beradi.

Kitob Misoli 15.6.1 — To'g'ri parallelepiped

∭_B xyz² dV, B = {0≤x≤1, −1≤y≤2, 0≤z≤3}

  1. x bo'yicha avval:
    ∫₀¹ xyz² dx = [x²yz²/2]₀¹ = yz²/2
  2. y bo'yicha:
    ∫_{-1}^2 yz²/2 dy = [y²z²/4]_{-1}^2 = (4z²/4) − (z²/4) = 3z²/4
  3. z bo'yicha:
    ∫₀³ 3z²/4 dz = [3z³/12]₀³ = [z³/4]₀³ = 27/4
Javob: 27/4
Kitob Misoli — Umumiy soha ustida triple integral

∭_E z dV, E = {0≤y≤√(1−x²), 0≤z≤√(1−x²)}, 0≤x≤1

E — x² + y² ≤ 1, x² + z² ≤ 1 ning birinchi oktantdagi kesishmasi

  1. Chegaralarni aniqlaymiz: 0≤x≤1, 0≤y≤√(1−x²), 0≤z≤√(1−x²)
  2. z bo'yicha:
    ∫₀^{√(1−x²)} z dz = [(1−x²)/2]
  3. y bo'yicha:
    ∫₀^{√(1−x²)} (1−x²)/2 dy = (1−x²)/2 · √(1−x²) = (1−x²)^{3/2}/2
  4. x bo'yicha (x = sinθ):
    ∫₀¹ (1−x²)^{3/2}/2 dx = (1/2) ∫₀^{π/2} cos⁴θ dθ = (1/2)·(3π/16) = 3π/32
Javob: 3π/32
15.7

Silindrli Koordinatlar

Nazariya

Silindrli koordinata o'zgarishi

Silindrli koordinatlar
x = r cosθ, y = r sinθ, z = z dV = r dr dθ dz x² + y² = r²

Silindrlar, konuslar, paraboloidlar bilan ishlashda ideal. Polar koordinataning 3D ko'rinishi.

Kitob Misoli 15.7.4 — Massa hisoblash

E: x²+y²≤1, z∈[1−r², 4], z-o'qiga masofa proporsional zichlik

ρ(x,y,z) = K√(x²+y²) = Kr

  1. Silindrli koordinatada E: 0≤θ≤π, 0≤r≤1, 1−r²≤z≤4
  2. Massa integrali:
    m = ∭_E K√(x²+y²) dV = ∫₀^π ∫₀¹ ∫_{1−r²}^4 Kr · r dz dr dθ
  3. z bo'yicha:
    ∫_{1−r²}^4 dz = 4 − (1−r²) = 3 + r²
  4. r va θ bo'yicha:
    = K ∫₀^π dθ · ∫₀¹ r²(3+r²) dr = Kπ [r³ + r⁵/5]₀¹ = Kπ(1 + 1/5) = 6Kπ/5
Javob: m = 6Kπ/5
Kitob Misoli 15.7.5 — Kartezian → silindrli

∫_{-2}^2 ∫_{-√(4−x²)}^{√(4−x²)} ∫_{√(x²+y²)}^2 (x²+y²) dz dy dx

  1. E sohasini aniqlang: E = {x²+y²≤4, √(x²+y²)≤z≤2} — konus bilan tekislik orasidagi soha
  2. Silindrli koordinatada: 0≤θ≤2π, 0≤r≤2, r≤z≤2
  3. Integral yozamiz:
    ∫₀^{2π} ∫₀² ∫_r^2 r² · r dz dr dθ = ∫₀^{2π} dθ · ∫₀² r³(2−r) dr
  4. r bo'yicha:
    ∫₀² r³(2−r) dr = ∫₀² (2r³ − r⁴) dr = [r⁴/2 − r⁵/5]₀² = 8 − 32/5 = 8/5
  5. θ bo'yicha: 2π · 8/5 = 16π/5
Javob: 16π/5
15.8

Sferik Koordinatlar

Nazariya

Sferik koordinata o'zgarishi

Sferik koordinatlar
x = ρ sinφ cosθ y = ρ sinφ sinθ z = ρ cosφ dV = ρ² sinφ dρ dφ dθ ρ² = x² + y² + z²
Kitob Misoli — Sfera hajmi

B: x²+y²+z² ≤ a² sferaning hajmini toping

  1. Sferik chegaralar: 0≤ρ≤a, 0≤φ≤π, 0≤θ≤2π
  2. Integral yozamiz:
    V = ∫₀^{2π} ∫₀^π ∫₀^a ρ² sinφ dρ dφ dθ
  3. ρ bo'yicha: [ρ³/3]₀^a = a³/3
  4. φ bo'yicha: ∫₀^π sinφ dφ = [−cosφ]₀^π = 2
  5. θ bo'yicha: ∫₀^{2π} dθ = 2π
  6. Jami: V = a³/3 · 2 · 2π = 4πa³/3
Javob: V = (4/3)πa³ ✓
Kitob Misoli — Sfera ρ=2 va konus z=√(x²+y²) orasidagi soha

∭_E z dV hisoblang

  1. φ₀ topish: konusda z = r, ya'ni ρcosφ = ρsinφ → tanφ = 1 → φ₀ = π/4
  2. Chegaralar: 0≤θ≤2π, 0≤φ≤π/4, 0≤ρ≤2
  3. z = ρcosφ ni qo'yamiz:
    ∭_E z dV = ∫₀^{2π} ∫₀^{π/4} ∫₀² (ρcosφ)·ρ² sinφ dρ dφ dθ
  4. ρ bo'yicha: ∫₀² ρ³ dρ = [ρ⁴/4]₀² = 4
  5. φ bo'yicha: ∫₀^{π/4} sinφ cosφ dφ = [sin²φ/2]₀^{π/4} = 1/4
  6. θ bo'yicha: ∫₀^{2π} dθ = 2π
  7. Jami: 4 · 1/4 · 2π = 2π
Javob: ∭_E z dV = 2π
16.2 — Birinchi Tur

Skalyar Chiziq Integrali

Nazariya

Chiziq integralining ta'rifi (§16.2)

Oddiy integraldan farqli, biz endi egri chiziq C bo'ylab integrallaymiz. C ni n ta kichik yoy Δsᵢ ga bo'lamiz va namunaviy nuqta P*ᵢ tanlaymiz.

Chiziq integral ta'rifi
∫_C f(x,y) ds = lim_{n→∞} ∑ᵢ f(x*ᵢ, y*ᵢ) Δsᵢ
Parametrik formula (hisoblash uchun)
∫_C f(x,y) ds = ∫_a^b f(x(t), y(t)) √[(x'(t))² + (y'(t))²] dt
3D (fazoviy egri chiziq) uchun
∫_C f(x,y,z) ds = ∫_a^b f(r(t)) |r'(t)| dt bu yerda |r'(t)| = √[(x')² + (y')² + (z')²]
Muhim xossalar

✓ Yo'nalishga bog'liq emas: ∫_{-C} f ds = ∫_C f ds (C ni teskari yo'nalsak ham javob o'zgarmaydi)

✓ Geometrik ma'no: f ≥ 0 bo'lsa, bu "to'siq"ning yuzasini ifodalaydi

✓ Massa: ρ(x,y) zichlik bo'lsa, m = ∫_C ρ ds

Kitob Misoli 16.2.1 — Yarim doira

∫_C (2 + x²y) ds, C: x² + y² = 1 ning yuqori yarmi

  1. Parametrlash: x = cos t, y = sin t, 0 ≤ t ≤ π
  2. ds ni topamiz:
    ds = √[(−sin t)² + (cos t)²] dt = √[sin²t + cos²t] dt = dt
  3. Integrandni almashtirish: 2 + x²y = 2 + cos²t · sin t
  4. Hisoblash:
    ∫₀^π (2 + cos²t sin t) dt = [2t − cos³t/3]₀^π = (2π + 1/3) − (0 − 1/3) = 2π + 2/3
Javob: 2π + 2/3
Kitob Misoli 16.2.2 — Bo'lakli egri chiziq

∫_C 2x ds, C = C₁ (parabola y=x², 0→1) ∪ C₂ (vertikal kesma, (1,1)→(1,2))

  1. C₁ da: x = x, y = x², 0 ≤ x ≤ 1
    ∫_{C₁} 2x ds = ∫₀¹ 2x √(1 + 4x²) dx = [1/4 · (2/3)(1+4x²)^{3/2}]₀¹ = (1/6)[(5)^{3/2} − 1] = (5√5 − 1)/6
  2. C₂ da: x = 1, y = y, 1 ≤ y ≤ 2
    ∫_{C₂} 2x ds = ∫₁² 2(1) √(0 + 1) dy = ∫₁² 2 dy = 2
  3. Jami: (5√5 − 1)/6 + 2 ≈ (5·2.236 − 1)/6 + 2 ≈ 1.731 + 2
Javob: (5√5 − 1)/6 + 2
Kitob Misoli 16.2.5 — Fazoviy spiral (Helix)

∫_C y sin z ds, C: x=cos t, y=sin t, z=t, 0 ≤ t ≤ 2π (spiral)

  1. |r'(t)| topamiz:
    r'(t) = (−sin t, cos t, 1) |r'(t)| = √(sin²t + cos²t + 1) = √2
  2. Integrandni almashtirish: y sin z = sin t · sin t = sin²t
  3. Hisoblash:
    ∫₀^{2π} sin²t · √2 dt = √2 ∫₀^{2π} (1 − cos 2t)/2 dt = √2 · [t/2 − sin 2t/4]₀^{2π} = √2 · π
Javob: √2 π
16.2 — Ikkinchi Tur

Vektor Chiziq Integrali — Ish (Work)

Nazariya

Vektor maydon bo'yicha chiziq integrali (§16.2)

F = ⟨P, Q, R⟩ kuch maydoni, C egri chiziq bo'lsin. Zarracha C bo'ylab harakat qilganda bajarilgan ish:

Kuch maydoni bo'yicha chiziq integrali (ish)
W = ∫_C F·dr = ∫_a^b F(r(t)) · r'(t) dt
Komponent ko'rinishi
∫_C F·dr = ∫_C P dx + Q dy + R dz
Muhim xossalar

✗ Yo'nalishga BOG'LIQ: ∫_{-C} F·dr = −∫_C F·dr

✓ Yo'nalishsiz (ds) bilan adashtirmang: ds ≥ 0 doim, lekin dr yo'nalishga bog'liq

Kitob Misoli 16.2.7 — Chorak doirada ish

F(x,y) = x²i − xy j, C: r(t) = cos t i + sin t j, 0 ≤ t ≤ π/2

  1. F(r(t)) ni topamiz:
    F(r(t)) = cos²t i − cos t sin t j
  2. r'(t) topamiz:
    r'(t) = −sin t i + cos t j
  3. Skalyar ko'paytma:
    F(r(t)) · r'(t) = cos²t·(−sin t) + (−cos t sin t)·cos t = −cos²t sin t − cos²t sin t = −2cos²t sin t
  4. Integral hisoblash:
    W = ∫₀^{π/2} (−2cos²t sin t) dt = [2cos³t/3]₀^{π/2} = 0 − 2/3 = −2/3
Javob: W = −2/3 (maydon harakatga qarshilik ko'rsatadi)
Kitob Misoli 16.2.8 — Twisted Cubic

∫_C F·dr, F(x,y,z) = xy i + yz j + zx k, C: r(t) = t i + t² j + t³ k, 0≤t≤1

  1. F(r(t)):
    F(r(t)) = t·t² i + t²·t³ j + t³·t k = t³ i + t⁵ j + t⁴ k
  2. r'(t) = i + 2t j + 3t² k
  3. F·r':
    t³·1 + t⁵·2t + t⁴·3t² = t³ + 2t⁶ + 3t⁶ = t³ + 5t⁶
  4. Hisoblash:
    ∫₀¹ (t³ + 5t⁶) dt = [t⁴/4 + 5t⁷/7]₀¹ = 1/4 + 5/7 = 27/28
Javob: 27/28
16.3

Chiziq Integrallar uchun Asos Teoremasi

Asosiy Teorema

Fundamental Theorem for Line Integrals (§16.3)

Agar F = ∇f konservativ maydon va C — a dan b gacha uzluksiz egri chiziq bo'lsa:

Chiziq Integrallar uchun Asos Teoremasi
∫_C ∇f · dr = f(r(b)) − f(r(a))

Bu degani: integral faqat bosh va oxirgi nuqtaga bog'liq, yo'l muhim emas!

Konservativ maydon belgilari

2D da: F = ⟨P,Q⟩ konservativ ⟺ ∂P/∂y = ∂Q/∂x

3D da: F konservativ ⟺ curl F = 0

Natijalar

1. Yo'ldan mustaqillik: ∫_{C₁} = ∫_{C₂} (bir xil uchlar)

2. Yopiq yo'l uchun: ∮_C F·dr = 0

Usul

Potensial funksiya f topish usuli

F = ⟨P, Q, R⟩ berilgan, ∇f = F bo'lgan f ni topish:

  1. fx = P dan boshlang: f = ∫P dx + g(y,z)
  2. fy = Q dan g ni aniqlang: g_y = Q − (f ning y ga hosilasi)
  3. fz = R dan h(z) ni aniqlang
  4. Tekshiring: ∇f = F bo'lishini tasdiqlang
Kitob Misoli 16.3.1 — Gravitatsion ish

F(x) = −mMG·x/|x|³ — (3,4,12)→(2,2,0) gacha bajarilgan ishni toping

  1. F = ∇f ekanligi ma'lum, bunda:
    f(x,y,z) = mMG / √(x² + y² + z²)
  2. FTLI qo'llaymiz:
    W = f(2,2,0) − f(3,4,12) = mMG/√(4+4+0) − mMG/√(9+16+144) = mMG/√8 − mMG/13 = mMG(1/(2√2) − 1/13)
Javob: W = mMG(1/(2√2) − 1/13) — yo'lni hisoblashga hojat yo'q!
Misol — Potensial funksiya va FTLI

F = ⟨2xy+z², x²+2yz, 2xz+y²⟩ uchun (0,0,0)→(1,2,3) integralini toping

  1. Konservativligini tekshirish (curl F = 0):
    ∂R/∂y − ∂Q/∂z = 2y − 2y = 0 ✓ ∂P/∂z − ∂R/∂x = 2z − 2z = 0 ✓ ∂Q/∂x − ∂P/∂y = 2x − 2x = 0 ✓
  2. fx = 2xy+z² → f = x²y + xz² + g(y,z)
  3. fy = x² + gz = x²+2yz → gz = 2yz → g = y²z + h(z)
  4. fz = 2xz + y² ✓ → f(x,y,z) = x²y + xz² + y²z
  5. FTLI:
    ∫_C F·dr = f(1,2,3) − f(0,0,0) f(1,2,3) = 1·2 + 1·9 + 4·3 = 2 + 9 + 12 = 23 f(0,0,0) = 0
Javob: 23
16.4

Green Teoremasi

Green Teoremasi (§16.4)

Green Teoremasi

C — musbat yo'nalgan (soat yelisi bo'yicha qarama-qarshi), yopiq, bo'lakli-tekis egri chiziq bo'lsin; D — C bilan chegaralangan soha.

Green Teoremasi
∮_C P dx + Q dy = ∬_D (∂Q/∂x − ∂P/∂y) dA
Green Teoremasining foydasi

✓ Murakkab chiziq integralni oddiy qo'sh integralga aylantirish

✓ Soha yuzasini hisoblash: A(D) = (1/2) ∮_C (x dy − y dx)

✓ Konservativ maydon tekshiruvi: ∂P/∂y = ∂Q/∂x ⟺ F konservativ

Kitob Misoli 16.4 — Green bilan soddalashtirish

∮_C (x⁴ + y) dx − (x + y⁴) dy, C: ellips x² + 4y² = 4 (soat yelisi bo'yicha qarama-qarshi)

  1. P = x⁴+y, Q = −x−y⁴ deb belgilaymiz
  2. ∂Q/∂x − ∂P/∂y = −1 − 1 = −2
  3. Green Teoremasi qo'llaymiz:
    ∮_C = ∬_D (−2) dA = −2 · A(D)
  4. Ellips yuzasini topamiz: x² + 4y² ≤ 4 → x²/4 + y²/1 ≤ 1, demak A = π·2·1 = 2π
  5. Javob: −2 · 2π = −4π
Javob: −4π
Misol — Green bilan yuza hisoblash

Uchburchak {(0,0), (3,0), (0,4)} yuzasini Green orqali toping

  1. Formula: A = (1/2)|∮_C (x dy − y dx)|
  2. Uchburchak tomonlari bo'yicha parameterlaymiz:

    C₁: (0,0)→(3,0): x=3t, y=0, dx=3dt, dy=0 → integral = 0

    C₂: (3,0)→(0,4): x=3−3t, y=4t, dx=−3dt, dy=4dt

    C₃: (0,4)→(0,0): x=0, y=4−4t, dx=0, dy=−4dt → integral = 0

  3. C₂ bo'yicha:
    ∫₀¹ [(3−3t)·4 − 4t·(−3)] dt = ∫₀¹ (12 − 12t + 12t) dt = ∫₀¹ 12 dt = 12
  4. A = (1/2)|12| = 6
Javob: A = 6 (tekshirish: (1/2)·3·4 = 6 ✓)
16.7

Yuza Integrallari — Surface Integrals

Nazariya — Skalyar Yuza Integrali

Skalyar yuza integrali (§16.7)

S: z = g(x,y) yuza bo'lsin, D — S ning xy tekisligiga proektsiyasi:

Skalyar yuza integrali
∬_S f(x,y,z) dS = ∬_D f(x,y,g(x,y)) √(1 + (∂z/∂x)² + (∂z/∂y)²) dA
Vektor yuza integrali (Flux) — z=g(x,y), yuqoriga yo'nalgan normal
∬_S F·dS = ∬_D F·⟨−g_x, −g_y, 1⟩ dA
Yuza maydoni formulasi
A(S) = ∬_D √(1 + (∂z/∂x)² + (∂z/∂y)²) dA
Kitob Misoli 16.7.2 — Skalyar yuza integrali

∬_S y dS, S: z = x + y², 0≤x≤1, 0≤y≤2

  1. ∂z/∂x = 1, ∂z/∂y = 2y → √(1+1+4y²) = √(2+4y²)
  2. Formula qo'llaymiz:
    ∬_S y dS = ∬_D y √(2+4y²) dA = ∫₀¹ dx · ∫₀² y √(2+4y²) dy
  3. ∫₀¹ dx = 1
  4. y bo'yicha (u = 2+4y²):
    ∫₀² y √(2+4y²) dy = [1/12 · (2+4y²)^{3/2}]₀² = (1/12)(18^{3/2} − 2^{3/2}) = (1/12)(54√2 − 2√2) = (52√2)/12 = 13√2/3
Javob: 13√2/3
Kitob Misoli — Flux hisoblash

F = ⟨y, x, z⟩, S: z = 1+x+y, 0≤x≤1, 0≤y≤1 (yuqoriga yo'nalgan) — fluxni toping

  1. g(x,y) = 1+x+y, g_x = 1, g_y = 1
  2. Flux formulasi:
    ∬_S F·dS = ∬_D F·⟨−1, −1, 1⟩ dA
  3. F·⟨−1,−1,1⟩ = −y − x + z = −y − x + (1+x+y) = 1
  4. Hisoblash:
    ∬_D 1 dA = A(D) = 1·1 = 1
Javob: Flux = 1
Kitob Misoli 16.7.3 — Bo'lakli yuza

∬_S z dS, S = S₁ (silindр x²+y²=1) ∪ S₂ (z=0 disk) ∪ S₃ (z=1+x tekislik qismi)

  1. S₁ uchun (θ, z parametrik):
    r(θ,z) = (cosθ, sinθ, z), 0≤θ≤2π, 0≤z≤1+cosθ |r_θ × r_z| = 1 ∬_{S₁} z dS = ∫₀^{2π} ∫₀^{1+cosθ} z dz dθ = ∫₀^{2π} (1+cosθ)²/2 dθ = 3π/2
  2. S₂ uchun (z=0): ∬_{S₂} z dS = ∬ 0 dS = 0
  3. S₃ uchun (z=1+x, polar da):
    ∬_{S₃} z dS = ∬_D (1+x) √2 dA = √2 ∫₀^{2π} ∫₀¹ (1+r cosθ) r dr dθ = √2 π
  4. Jami: 3π/2 + 0 + √2 π
Javob: 3π/2 + √2 π
16.8

Stokes Teoremasi

Stokes Teoremasi (§16.8)

Stokes Teoremasi

S — oriyentlangan tekis-bo'lakli yuza, C = ∂S — S ning musbat yo'nalgan chegarasi:

Stokes Teoremasi
∮_C F·dr = ∬_S (curl F)·dS
curl F (rotor) hisoblash
curl F = |i j k | |∂/∂x ∂/∂y ∂/∂z| |P Q R | = (R_y−Q_z)i − (R_x−P_z)j + (Q_x−P_y)k
Stokes Teoremasining foydasi

✓ Murakkab yuzani oddiy yuzaga almashtirish (bir xil chegara = bir xil integral)

✓ Murakkab chiziq integralni yuzaviy integralga aylantirish

✓ F konservativ ⟺ curl F = 0 ⟺ ∮ F·dr = 0

Kitob Misoli 16.8.1 — Ellips bo'yicha integral

∮_C F·dr, F = ⟨−y², x, z²⟩, C: y+z=2 tekisligi va x²+y²=1 silindri kesishmasi

  1. curl F hisoblash:
    curl F = (0−0)i − (0−0)j + (1−(−2y))k = (1+2y)k
  2. Qulay yuza tanlang: S = elliptik D soha y+z=2 tekisligida, proyeksiya: x²+y²≤1
  3. Yuqoriga normal: z = 2−y, g_x=0, g_y=−1
  4. Stokes qo'llaymiz:
    ∬_D curl F · ⟨−g_x, −g_y, 1⟩ dA = ∬_D (1+2y)·1 dA = ∫₀^{2π} ∫₀¹ (1 + 2r sinθ) r dr dθ
  5. Hisoblash:
    = ∫₀^{2π} [r²/2 + 2r³ sinθ/3]₀¹ dθ = ∫₀^{2π} (1/2 + 2sinθ/3) dθ = π + 0 = π
Javob: π
Kitob Misoli 16.8.2 — Yuza integralni soddalashtirish

∬_S (curl F)·dS, F=⟨xz, yz, xy⟩, S: x²+y²+z²=4 ning x²+y²≤1 va z>0 qismi

  1. Chegara C: x²+y²=1, z=√3 doirasi
    r(t) = (cos t, sin t, √3), r'(t) = (−sin t, cos t, 0), 0≤t≤2π
  2. F(r(t)) = (√3 cos t, √3 sin t, cos t sin t)
  3. F·r':
    F·r' = −√3 cos t sin t + √3 sin t cos t + 0 = 0
  4. Stokes:
    ∬_S (curl F)·dS = ∮_C F·dr = ∫₀^{2π} 0 dt = 0
Javob: 0
16.9

Divergence Teoremasi

Gauss-Ostrogradskiy Teoremasi (§16.9)

Divergence (Gauss) Teoremasi

E — yopiq qattiq jism, S = ∂E — E ning tashqariga yo'nalgan normal bilan chegarasi:

Divergence Teoremasi
∯_S F·dS = ∭_E div F dV
div F hisoblash
div F = ∂P/∂x + ∂Q/∂y + ∂R/∂z
Fizik ma'nosi

div F > 0: nuqta "manba" (source) — oqim chiqadi

div F < 0: nuqta "lavha" (sink) — oqim kiradi

div F = 0: siqilmas suyuqlik

Kitob Misoli 16.9.1 — Birlik sfera flux

F = ⟨z, y, x⟩, S: x²+y²+z²=1 — tashqi fluxni toping

  1. div F = ∂z/∂x + ∂y/∂y + ∂x/∂z = 0 + 1 + 0 = 1
  2. Divergence Teoremasi:
    ∯_S F·dS = ∭_B 1 dV = V(B) = (4/3)π(1)³ = 4π/3
Javob: 4π/3
Kitob Misoli 16.9.2 — Murakkab maydon, oddiy divergensiya

F = ⟨xy, y²+e^{xz²}, sin(xy)⟩, E = {z=1−x², z=0, y=0, y+z=2} bilan chegaralangan

  1. div F:
    ∂(xy)/∂x + ∂(y²+e^{xz²})/∂y + ∂sin(xy)/∂z = y + 2y + 0 = 3y
  2. E sohasini yozamiz: −1≤x≤1, 0≤z≤1−x², 0≤y≤2−z
  3. Divergence Teoremasi:
    ∯_S F·dS = ∭_E 3y dV = 3 ∫_{-1}^1 ∫₀^{1−x²} ∫₀^{2−z} y dy dz dx
  4. y bo'yicha: ∫₀^{2−z} y dy = (2−z)²/2
  5. z bo'yicha:
    ∫₀^{1−x²} (2−z)²/2 dz = [−(2−z)³/6]₀^{1−x²} = −(1+x²)³/6 + 4/3
  6. x bo'yicha (simmetriya):
    3 ∫_{-1}^1 (4/3 − (1+x²)³/6) dx = 184/35
Javob: 184/35

∮_C P dx + Q dy = ∬_D (Q_x − P_y) dA
