01 / Chiziq Integrali

Line Integrals

Nazariya

Chiziq integrali nima?

Chiziq integrali — bu egri chiziq bo'ylab hisoblangan integral. Oddiy integraldan farqi shuki, biz to'g'ri chiziq bo'yicha emas, balki ixtiyoriy egri chiziq C bo'yicha integrallashtiramiz.

Fizikada: zaryadlangan zarraga kuch maydoni orqali ish hisoblash, massali sim og'irlik markazi — bular chiziq integrali.

Arc length bo'yicha (yo'nalishga bog'liq EMAS)
∫_C f(x,y) ds
Vektor maydon bo'yicha (yo'nalishga BOG'LIQ)
∫_C F·dr = ∫_C P dx + Q dy + R dz
Parametrik holda hisoblash (r(t) = ⟨x(t), y(t)⟩)
∫_C F·dr = ∫_a^b F(r(t)) · r'(t) dt
Arc length elementi
ds = √(x'(t)² + y'(t)²) dt
▶ Misol 1 — Sodda chiziq integrali

∫_C (x+y) ds, bu yerda C: x²+y²=1 ning birinchi chorak qismi

C ni parametrik: x=cos t, y=sin t, 0≤t≤π/2

  1. ds topamiz: ds = √(x'²+y'²) dt = √(sin²t + cos²t) dt = dt
  2. Integrandni almashtirish: x+y = cos t + sin t
  3. Hisoblash: ∫₀^{π/2} (cos t + sin t) dt = [sin t − cos t]₀^{π/2}
  4. Javob: = (1−0) − (0−1) = 2
▶ Misol 2 — Exam masalasi (Part II)

F = ⟨2xy+z², x²+2yz, 2xz+y²⟩, A=(0,0,0) → B=(1,2,3)

  1. curl F = 0 ni tekshirish:
    i: ∂(2xz+y²)/∂y − ∂(x²+2yz)/∂z = 2y − 2y = 0 ✓
    j: ∂(2xy+z²)/∂z − ∂(2xz+y²)/∂x = 2z − 2z = 0 ✓
    k: ∂(x²+2yz)/∂x − ∂(2xy+z²)/∂y = 2x − 2x = 0 ✓
  2. Potensial funksiya f topish:
    fx = 2xy+z² → f = x²y + xz² + g(y,z)
    fy = x² + gz = x²+2yz → gz = 2yz → g = y²z + h(z)
    fz = 2xz + y² ✓
    ∴ f(x,y,z) = x²y + xz² + y²z
  3. FT for Line Integrals:
    ∫_C F·dr = f(B) − f(A)
    f(1,2,3) = 1·2 + 1·9 + 4·3 = 2+9+12 = 23
    f(0,0,0) = 0
    Javob: 23
Muhim qoidalar

Konservativ maydon belgilari

✓ 2D da: F = ⟨P,Q⟩ konservativ ⟺ ∂P/∂y = ∂Q/∂x

✓ 3D da: F = ⟨P,Q,R⟩ konservativ ⟺ curl F = 0

Natija 1: ∫_C F·dr = f(B) − f(A) — faqat uchki nuqtalarga bog'liq

Natija 2: Har qanday yopiq C uchun ∮_C F·dr = 0

02 / Yuza Integrali

Surface Integrals

Nazariya

Yuzaviy integral nima?

Yuzaviy integral — bu ikki o'lchovli yuza orqali o'tadigan oqimni (flux) hisoblaydi. Fizikada: elektr maydon oqimi, suyuqlik oqimi, issiqlik oqimi.

Ikki turi bor: skalyar yuzaviy integral (massa, yuza maydoni) va vektorial yuzaviy integral (flux).

Skalyar yuza integrali (z = f(x,y) bo'lsa)
∬_S g dS = ∬_D g(x,y,f) √(1 + fx² + fy²) dA
Flux (vektor maydon, yuqoriga yo'nalgan normal)
∬_S F·dS = ∬_D F · ⟨−fx, −fy, 1⟩ dA
Yopiq yuza uchun (Divergence Theorem)
∯_S F·dS = ∭_E div F dV
▶ Misol 1 — Yuza maydoni

z = √(x²+y²), x²+y²≤4 yuza maydonini toping

  1. fx va fy topamiz: fx = x/√(x²+y²), fy = y/√(x²+y²)
  2. √(1+fx²+fy²): = √(1 + x²/(x²+y²) + y²/(x²+y²)) = √2
  3. Polar koordinat: A = ∬_D √2 dA = √2 · π·4 = 4π√2
▶ Misol 2 — Exam masalasi (Part III): Divergence Theorem

F = (x, y, z), S = paraboloid + disk, tashqi flux toping

  1. div F: ∂x/∂x + ∂y/∂y + ∂z/∂z = 1+1+1 = 3
  2. Div Theorem qo'llaymiz: ∯F·dS = ∭_E 3 dV = 3·V(E)
  3. Silindr koordinatada V(E) hisoblash:
    V = ∫₀^{2π} ∫₀^2 ∫₀^{4−r²} r dz dr dθ
    = 2π ∫₀^2 r(4−r²) dr
    = 2π [2r² − r⁴/4]₀^2
    = 2π (8 − 4) = 8π
  4. Flux = 3 · 8π = 24π
  5. Tekshirish: S₁ fluxsi = 24π, S₂ fluxsi = 0 (chunki z=0 → F·n=0). Jami: 24π ✓
03 / Uch O'lchovli Integral

Triple Integrals

Koordinat Sistemalari

Qaysi koordinatani tanlash kerak?

Sistema Formulalar dV Qachon ishlatish
Kartezian x, y, z dx dy dz To'g'ri to'rtburchak soha
Silindrli x=r cosθ, y=r sinθ, z=z r dr dθ dz Silindrik simmetriya, konus
Sferik x=ρsinφcosθ, y=ρsinφsinθ, z=ρcosφ ρ² sinφ dρ dφ dθ Sfera, yarim sharlar
▶ Misol 1 — Exam masalasi (Part IV-b): Hajm

E = sfera ρ=2 va konus z=√(x²+y²) orasidagi soha hajmi

  1. Chegaralar: 0≤θ≤2π, 0≤φ≤π/4, 0≤ρ≤2
  2. Integral yozish:
    V = ∫₀^{2π} ∫₀^{π/4} ∫₀^2 ρ² sinφ dρ dφ dθ
  3. ρ bo'yicha: ∫₀^2 ρ² dρ = [ρ³/3]₀^2 = 8/3
  4. φ bo'yicha: ∫₀^{π/4} sinφ dφ = [−cosφ]₀^{π/4} = 1 − √2/2 = (2−√2)/2
  5. θ bo'yicha: ∫₀^{2π} dθ = 2π
  6. Jami: V = (8/3) · (2−√2)/2 · 2π = 8π(2−√2)/3
▶ Misol 2 — Exam masalasi (Part IV-c,d): Sentroid

∭_E z dV va z̄ (sentroid) hisoblash

  1. z = ρcosφ ni qo'yamiz:
    ∭_E z dV = ∫₀^{2π} ∫₀^{π/4} ∫₀^2 (ρcosφ)·ρ² sinφ dρ dφ dθ
  2. ρ bo'yicha: ∫₀^2 ρ³ dρ = [ρ⁴/4]₀^2 = 4
  3. φ bo'yicha: ∫₀^{π/4} sinφ cosφ dφ = [sin²φ/2]₀^{π/4} = 1/4
  4. θ bo'yicha: 2π
  5. ∭z dV = 4 · 1/4 · 2π = 2π
  6. Sentroid:
    z̄ = ∭z dV / V = 2π / [8π(2−√2)/3]
    = 3 / [4(2−√2)]
    = 3(2+√2) / 8 ≈ 1.28
    Geometrik ma'no: qattiq jismning og'irlik markazi z=1.28 da joylashgan. Konus pastki qismni "kesib" olgani uchun massa yuqoriroqda to'plangan.
04 / Teoremalar

To'rtta Buyuk Teorema

Asosiy g'oya

Hammasi bir tamoyildan

Bu to'rtta teorema bir g'oyaning turli o'lchamli ko'rinishi: "Chegaradagi integral = ichki hududdagi integral". Har bir teorema bir o'lcham "yuqori" ko'taradi.

Teorema 01

FT for Line Integrals

∫_C ∇f · dr
= f(r(b)) − f(r(a))

Konservativ maydonda integral faqat bosh va oxirgi nuqtaga bog'liq. Yo'l muhim emas.

Nuqta → Nuqta

Teorema 02

Green's Theorem

∮_C P dx + Q dy
= ∬_D (∂Q/∂x − ∂P/∂y) dA

Yopiq egri chiziq bo'yicha integral — uni o'rab turgan D hudud bo'yicha integralga aylanadi.

Egri chiziq → Yuza (2D)

Teorema 03

Stokes' Theorem

∮_C F · dr
= ∬_S (curl F) · dS

Green teoremaning 3D umumlashmasi. C chegarasi S yuzaning chegarasi.

Egri chiziq → Yuza (3D)

Teorema 04

Divergence Theorem

∯_S F · dS
= ∭_E div F dV

Yopiq yuza orqali oqim — ichidagi hududdagi divergensiyaga teng.

Yuza → Hajm

Muhim identikliklar

Doim to'g'ri bo'lgan formulalar

div(curl F) = 0
har qanday F uchun
curl(∇f) = 0
har qanday f uchun
∮_C F·dr = 0
F konservativ bo'lsa
∂P/∂y = ∂Q/∂x
2D konservativ sharti
05 / Test

O'z-o'zini Tekshirish

1. Fubini teoremasi qaysi tenglikni bildiradi?
A Qo'sh integral ikkita oddiy integralning ko'paytmasiga teng
B ∬_R f dA = ∫_a^b ∫_c^d f dy dx = ∫_c^d ∫_a^b f dx dy
C Qo'sh integral faqat f≥0 bo'lganda mavjud
D Ichki integral doim y bo'yicha hisoblangishi kerak
✓ To'g'ri. Fubini teoremasi: uzluksiz f uchun ikki yo'nalishdagi takrorlangan integral teng va ikkala tartibda ham hisoblanishi mumkin.
2. Polar koordinatada dA = dx dy ning o'rniga nima yoziladi?
A dr dθ
B r dr dθ
C r² dr dθ
D sin θ dr dθ
✓ To'g'ri. Yakobiani r, shuning uchun dA = r dr dθ. Bu "r" ni unutmaslik juda muhim — polar koordinatada eng ko'p kechadigan xato.
3. F konservativ bo'lsa, yopiq C egri chiziq uchun ∮_C F·dr = ?
A 1
B ∞
C 0
D yo'lga bog'liq
✓ To'g'ri. F = ∇f bo'lsa, FT Line Integrals bo'yicha ∫_C F·dr = f(B)−f(A). Yopiq yo'l uchun A=B, shuning uchun f(B)−f(A)=0.
4. Sferik koordinatada dV ning formulasi qaysi?
A ρ dρ dφ dθ
B ρ² dρ dφ dθ
C ρ² sinφ dρ dφ dθ
D ρ² cosφ dρ dφ dθ
✓ To'g'ri. Sferik koordinata Yakobiani ρ²sinφ. sinφ ni unutmaslik kerak — bu φ = 0 va φ = π uchun nolga teng (qutb yo'nalishlarida kompressiya).
5. div(curl F) = ?
A ∇²F (vektorial Laplas)
B curl(div F)
C 0 (har qanday ikkinchi tartib uzluksiz F uchun)
D F konservativ bo'lsa 1, aks holda 0
✓ To'g'ri. Bu vektorial analizning muhim identikliklaridan biri: div(curl F) ≡ 0. Gidrodinamikada: "rotatsional oqimning divergensiyasi nol" degani.
6. F = ⟨y, x⟩ maydon konservativmi?
A Ha, chunki ∂P/∂y = ∂Q/∂x
B Yo'q, chunki curl F ≠ 0
C Yo'q, chunki div F ≠ 0
D Aniqlab bo'lmaydi
✓ To'g'ri. P=y, Q=x → ∂P/∂y = 1, ∂Q/∂x = 1. Teng, demak konservativ. Potensial funksiya: f(x,y) = xy (chunki ∂(xy)/∂x = y = P va ∂(xy)/∂y = x = Q).
